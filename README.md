# Agile/Scrum Лабораторная работа
## Проект: Система управления задачами
Простое веб-приложение для управления персональными задачами. 
Позволяет создавать, редактировать, отмечать выполнение и фильтровать задачи.
Данные сохраняются в браузере (LocalStorage).

### Технологии:
- HTML5, CSS3, JavaScript (чистый)
- LocalStorage для хранения данных
- Адаптивный дизайн

### Цель проекта:
Практика применения методологии Scrum через GitHub Issues.
